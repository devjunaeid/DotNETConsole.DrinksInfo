### Uses.
- #### Clone Repo.
  `git clone https://github.com/devjunaeid/DotNETConsole.Flashcards.git`
- #### Go to the project folder and restore.
    `dotnet restore`
- #### Build.
    `dotnet build`
- #### Run.
    `dotnet run`
